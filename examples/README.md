# Coding Examples 💻

This folder contains **example code from our daily coding demos**.

Students are encouraged to **open these examples and keep them on their screen during the demo**.

You can use them to:

- see what the code should look like  
- catch up if you fall behind  
- help fix bugs  
- check your work  

Getting stuck, falling behind, or breaking your code is **100% normal when learning to program**.

These examples are here to **help you keep going**.

## Important

Please **do NOT copy and paste the code**.

Instead:

- read the example  
- understand what it does  
- type the code yourself  

Typing the code helps your brain **learn how programming works**.

The goal is not to finish fast.

The goal is to **learn and try your best**. 🚀
