# ThinkPad Club 🌻

Welcome to the **ThinkPad Club Resources Repository**! This is the main place where we keep the code, project guides, and other fun resources we use in ThinkPad Club. You can come here to review demos, keep working on projects at home, explore new programming ideas, and find helpful tools we use in class like Python, p5.js, Scratch, and Google Colab. Think of this repo as your **club toolbox**—a place to learn, experiment, fix bugs, and get ideas for new things to build. Remember, programming is all about **trying things, breaking things, and trying again**, so explore, have fun, and don’t worry if things don’t work the first time!

## Our Class Motto

- **"Why are we here, ThinkPad Club?"**
- **"To focus, code and learn!"**

## Our Class Conduct Rules

During ThinkPad Club we want everyone to stay focused and respectful so the coding demonstrations and projects go smoothly. These rules help students concentrate on their work and make sure no one distracts others who are trying to learn.

1. **Listen and work during coding demonstrations**
2. **Wear headphones to focus on work after demonstrations**
4. **Stay in your seat and focus on your own project**  
5. **Raise your hand if you need a break, snack, or bathroom**  
6. **Work only on your own computer**  
7. **Keep hands to yourself at all times**  
8. **Use inside voices**
9. **Do not distract other students**
10. **No unapproved games, websites or videos**
