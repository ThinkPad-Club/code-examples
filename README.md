# ThinkPad Club – Code Examples

Welcome to the **ThinkPad Club Code Examples Repository**.

This repository contains code from **live demonstrations during ThinkPad Club meetings**. These examples are here to help students:

- Follow along during demonstrations
- Catch up if they fall behind
- Debug their own programs
- Restart from a working example
- Explore and experiment with code after class

---

## How to Use This Repository

Each folder contains a small project or example that was demonstrated during a club meeting.

Students can:

1. Open the example code
2. Compare it with the code they wrote during the demo
3. Copy parts of the code to fix bugs
4. Restart their project from the example if needed

These examples are meant to **support learning**, not replace the process of writing the code yourself.

---

## Running the Code

Most of the examples are meant to be run using tools we use in club, such as:

- Python
- p5.js
- Scratch
- Google Colab
- GitHub

If you are unsure how to run an example:

- Follow along with the original demonstration
- Ask for help during club
- Ask a mentor or instructor
