# Student Projects 🚀

This folder is where students can **upload their coding projects** from our demonstrations.

Projects can be **finished or unfinished**.  
What matters most is that you **try and share your work**.

---

# File Names

Files should always start with:

**your first name + the project name**

Examples:

```
alex_bouncing_ball.py
maya_scratch_game.sb3
liam_music_project.js
sofia_robot_drawing.py
```

This helps us keep everyone's projects organized.

---

# How to Upload Your Project

1. Open the **projects folder** in GitHub.
2. Click the **Add file** button near the top of the page.
3. Click **Upload files**.
4. Drag your project file into the upload box.
5. Make sure the **file name starts with your first name**.
6. Scroll down and click **Commit changes**.

Your project is now saved!

---

# Important

You can upload projects that are:

- finished
- unfinished
- still being worked on

All programmers share work that is **still in progress**.

The goal is to **learn, try new things, and keep building**. 💻🌻
